# Minds and Actions Learning Center

A modern, responsive website for Minds and Actions, a non-profit organization dedicated to empowering children and seniors through STEM education and digital literacy programs.

## 🌟 About

Minds and Actions bridges the digital divide by offering specialized programs for two key demographics:
- **Children**: STEM workshops, coding classes, robotics clubs, and programming courses
- **Seniors**: Digital literacy training, computer skills classes, tech support, and video calling assistance

## 🎯 Features

- **Age-Focused Programs**: Tailored content for children (STEM/coding) and seniors (digital literacy)
- **Interactive Calendar**: Browse and register for upcoming events and workshops
- **Event Details**: Comprehensive information about each program with registration capabilities
- **Community News**: Stay updated with achievements, program launches, and community milestones
- **Volunteer-Led**: Programs run by dedicated community volunteers
- **Free Access**: Most programs are offered at no cost to participants

## 🚀 Technology Stack

- **React 18** with TypeScript for robust component development
- **Vite** for fast development and building
- **Tailwind CSS** for modern, responsive styling
- **React Router DOM** for seamless navigation
- **Lucide React** for consistent iconography

## 📱 Pages

1. **Homepage**: Welcome message, program showcases, and upcoming events
2. **Activity Center**: Complete program listings with filtering by age group and category
3. **Calendar**: Interactive calendar view with event details and registration
4. **News**: Community updates, achievements, and program announcements
5. **Event Details**: In-depth information about specific programs and events

## 🎨 Design Philosophy

- Clean, accessible interface suitable for all ages
- Color-coded age groups (Green for children, Amber for seniors)
- No gradients policy for consistent, professional appearance
- Mobile-first responsive design
- Intuitive navigation and user experience

## 🤝 Community Impact

Minds and Actions serves the community by:
- Providing free STEM education to underserved children
- Helping seniors stay connected through technology
- Creating intergenerational learning opportunities
- Supporting volunteer educators and mentors
- Building digital literacy across all age groups
